全称IntelliJ IDEA
集成环境：把代码编写、编译、执行、调试等多种功能综合到一起的开发工具

IDEA启动的基本配置：
①创建一个空项目
②创建一个新模块
③在idea test模块下的src下创建一个包(com.–)
④在com.–包下新建一个类(HelloWorld) 在HelloWorld类中编写代码
⑤在idea中执行程序

设置字体：file → Setting → Editor → Font

内容辅助键和快捷键

①快速生成语句

快速生成 main() 方法：psvm,回车
快速生成输出语句：sout，回车

②内容辅助键

Ctrl + Alt + space(内容提示、代码补全等)

③快捷键

注释
单行：选中代码，Ctrl + /，再来一次就是取消
多行：选中代码，Ctrl + Shift + /，再来一次就是取消

Ctrl + Alt + L → 格式化，对齐代码（不用选中，直接用即可）
Ctrl + Alt + 空格 → 防止手误导致（快速生成语句等）快捷方式不能使用，注意光标要贴着
Ctrl + Alt + V → 快速生成变量对应的类

创建一个类之后，快速产生构造方法（有参、无参）
Alt + Insert →用于生成代码或者创建新的文件/类

生成 Getter 和 Setter 方法
生成构造函数——-选择”Constructor”选项。
生成重写方法——-“Override Methods”选项。
创建新的文件/类——-例如”New File”或者”New Class”。

Alt + Enter → 用于进行代码智能提示和修复
当你将光标放置在有错误或警告的代码处时，按下Alt + Enter，IDEA会弹出一个菜单，其中包含了修复该问题的建议。你可以通过选择菜单中合适的选项来进行代码修复。这个快捷键也可以用来快速修改代码并应用其他优化建议。
Ctrl + F → 查询（在查看源码时常用）
Ctrl + N → 可查询类名
