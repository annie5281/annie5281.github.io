名词解释

2022-05-15

URL

URL (Uniform Resource Locator)（统一资源定位符）
用于指定互联网上资源的地址。
在Android应用中，URL通常用于访问网络资源，比如网页、图片、视频等。
开发者可以使用 URL 类来处理和操作 URL ，比如打开连接、读取数据等操作。

URI

URI(Uniform Resource Identifier)（统一资源标识符）
是一个用于标识某一资源的字符串，它提供了一种统一的方式来定位资源。
在Android开发中，URI 通常用于表示文件、内容提供者、网页链接等资源的地址

URI由以下几部分组成：

  1. 协议部分：指定访问资源的协议，如http、https、file等。
  2. 主机部分：指定资源所在的主机名或IP地址。
  3. 端口部分：指定资源所在的端口号。
  4. 路径部分：指定资源在服务器上的路径。
  5. 查询部分：用于传递参数给服务器。
  6. 锚点部分：指定资源内部的位置。

在Android开发中， URI 常用于表示文件路径、ContentProvider 的 URI 、 Intent 的数据等。
例如，通过URI可以访问手机存储中的文件、访问 ContentProvider 提供的数据、打开网页链接等。
在 FileProvider 中，也使用 URI 来表示共享文件的地址。

UML

UML 是 Unified Modeling Language （统一建模语言）的缩写。
它是一种用于软件开发的标准化建模语言，用于描述、设计和分析软件系统的结构、行为和交互。
UML 提供了一组图形符号和规则，使开发人员能够更好地理解和沟通系统设计和实现的细节。

字节码文件

字节码文件是一种中间代码文件，它包含了用于在Java虚拟机（JVM）上执行的Java程序的指令集。
当Java源代码被编译成字节码文件时，它不会直接被翻译成机器码，而是转换成一种与特定平台无关的中间代码。
这样的设计使得Java程序具有跨平台性，可以在任何安装了Java虚拟机的平台上运行。

字节码文件通常以.class为扩展名，例如HelloWorld.class。
这些文件包含了Java程序的类定义、方法、变量以及其他相关信息。
当Java程序被执行时，JVM会将字节码文件加载到内存中，并逐条解释执行其中的指令。

字节码文件的存在使得Java程序具有平台无关性和安全性，因为字节码文件不能直接被执行，只能在Java虚拟机上运行。
这也使得Java程序更容易进行跨平台的开发和部署。

MIME

MIME 类型 (Multipurpose Internet Mail Extensions) 多用途互联网邮件扩展
是一种用于标识文件格式和互联网媒体类型的标准。
在Android开发中，图片的 MIME 类型用于表示图片文件的格式。
MIME 类型通常由两部分组成，第一部分表示大类别（如“image”表示图像文件），第二部分表示具体的子类别（如“jpeg”表示JPEG格式的图像文件）。

常见的图片MIME类型包括：

  • image/jpeg（JPEG格式）
  • image/png（PNG格式）
  • image/gif（GIF格式）
  • image/bmp（BMP格式）
  • image/webp（WebP格式）

根据图片文件的实际格式，可以通过查看文件的扩展名（如.jpg、.png等）来确定其对应的MIME类型。

富文本

富文本是一种可以包含文字、图像、视频、链接等多种元素，并可以进行丰富样式和格式设置的文本格式。
与普通的纯文本相比，富文本具有更强的表现力和交互性，常用于网页设计、编辑器、邮件等场景中。
常见的富文本编辑器包括 Microsoft Word 、 Google Docs 、 HTML编辑器 等。

Wiki 维基

维基 (Wiki) 是一种可以由多个用户共同编辑的基于网络的知识共享平台。
维基通常采用类似于网页编辑的方式，用户可以通过简单的编辑、添加、修改网页内容的途径，为平台上的文章、页面等内容做出贡献。
维基最著名的例子就是维基百科，是一个由志愿者共同编纂的免费在线百科全书。
由于维基的开放性和民主性，使得维基平台成为了一种强大的知识汇聚和分享工具。
